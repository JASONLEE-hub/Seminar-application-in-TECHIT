## 👉 プロジェクト紹介
> 開発期間 : 2023.09.04 ~ 2023.09.14 (10日間)
- TECHITが主催するセミナーイベントに関する情報を表示し、チケッティングができるようサポートするアプリ


<br/><br/>

## 👉 インストール / 実行方法
- Firebase Google Info消費者用、管理者用の設置が必要
```
GoogleService-info.plist
```

<br/><br/>

### 🎫🦁 消費者アプリ
<details>
<summary>セミナーリスト</summary>
  
  - 開催予定のセミナーリストを表示
  - セミナーの詳細情報を表示
    - セミナー名やテーマ、開催日時、開催場所など
  - セミナー参加申し込み

</details>  


<details>
<summary>私のセミナー</summary>

  - ユーザーがお気に入りのセミナー情報の表示、管理
  - ユーザーが参加申請したセミナー情報の表示
  - 参加申し込みしたセミナーキャンセル機能
  - ユーザーが最近見たセミナー情報表示
  
</details>  


<details>
<summary>設定</summary>

  - ログイン、ログアウト機能
  - ユーザー情報表示
  - 開発者情報ウェブビューで表示

</details>
